# CANONICAL ERROR CODE REGISTRY

## Purpose
Defines all system error codes. Codes are stable and machine-readable.

---

## Error Shape
{
  "error": {
    "code": "EXECUTION_ALREADY_CLAIMED",
    "message": "Operation is already claimed"
  }
}

Rules:
- code is authoritative
- message is non-authoritative

---

## Naming
<DOMAIN>_<ERROR_NAME>
