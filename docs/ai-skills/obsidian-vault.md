# Skill: obsidian-vault

## Purpose
Manage knowledge notes in an Obsidian-style vault using Markdown, wikilinks, and index notes.

## Use when
- Creating structured notes.
- Linking related concepts.
- Maintaining a knowledge base.

## Process
1. Identify note type:
   - concept
   - project
   - decision
   - meeting
   - reference
2. Create or update the note.
3. Add wikilinks to related notes.
4. Update index notes.
5. Keep note titles stable and searchable.

## Output format
```md
# <Note Title>

## Summary
## Details
## Links
- [[Related Note]]

## Next Actions
```

## Rules
- One note should have one clear purpose.
- Use wikilinks consistently.
- Do not duplicate content across notes; link instead.
