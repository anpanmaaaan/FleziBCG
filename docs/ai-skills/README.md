## FE/UI Skill Routing

Use `docs/ai-skills/stitch-design-md-ui-ux/SKILL.md` when work touches:

- frontend UI/UX
- React components
- Tailwind styling
- screen packs
- Figma Make
- Google Stitch
- `DESIGN.md`
- source-aligned UI implementation

This skill complements, but does not replace:

- `hard-mode-mom-v3`
- `qa-e2e-layer`
- `pr-gate-reviewer`
- `design-system-enforcer`

For execution, quality, material, station, operation, or governed-action UI, apply both `stitch-design-md-ui-ux` and `hard-mode-mom-v3`.
