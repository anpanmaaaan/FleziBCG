# Skill: edit-article

## Purpose
Improve an article, essay, or long-form text for clarity, structure, and flow.

## Use when
- Text is too verbose.
- Argument structure is unclear.
- The user wants a stronger article without changing meaning.

## Process
1. Identify the intended audience.
2. Identify the main argument.
3. Diagnose structure issues.
4. Improve section order.
5. Tighten wording.
6. Preserve author voice.
7. Return either:
   - edited version
   - change notes
   - both

## Output format
```md
# Edited Article

<edited text>

## Change Notes
- ...
```

## Rules
- Do not change factual meaning.
- Do not over-polish into generic corporate prose.
- Preserve concrete examples.
- Remove repetition.
