# Skill: scaffold-exercises

## Purpose
Create exercise directory structures with problems, solutions, explainers, and test scaffolding.

## Use when
- Building educational coding exercises.
- Creating workshop material.
- Splitting concepts into progressive tasks.

## Process
1. Define learning objectives.
2. Split into sections.
3. For each section, create:
   - Problem statement
   - Starting files
   - Tests
   - Solution
   - Explanation
4. Ensure each exercise teaches one concept.

## Output format
```md
# Exercise Scaffold

## Learning Objectives
## Sections
## Directory Structure
## Exercise List
## Test Strategy
## Solution Notes
```

## Rules
- Avoid exercises that require hidden knowledge.
- Keep tasks incremental.
- Make tests clear and educational.
